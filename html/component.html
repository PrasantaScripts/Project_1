IntAirSense

IntAirSense -
Intelligent Air Pollution Monitoring
for Smart Cities using Low Cost Sensors

Project Overview

Project Description

IndoAir aims to create a pre-trained model that can predict indoor PM2.5 and CO2 levels using sparse data from Air Quality Monitoring Devices placed outside classrooms. They use a simple regression model for PM2.5 and a Random Forest-based classifier for CO2. The predicted PM2.5 values are compartmentalized into 5 AQI classes, while CO2 values are classified into 2 classes using silhouette score and K-Means clustering. The proposed system provides air quality monitoring of indoor environments using a low-cost thermo-hygrometer with an accuracy of 78% for PM2.5 and 73% for CO2. The system provides a map view of the building and rooms where it is installed and triggers suggestions to reduce exposure to air pollutants. This work would help instructors/teachers maintain better air quality inside the classroom for the students' well-being.
